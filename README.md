## <a href="https://www.nuget.org/packages/DarkUI.CE"><img align="right" alt="NuGet Version" src="https://img.shields.io/nuget/vpre/DarkUI.CE?style=flat-square"></a> DarkUI Community Edition
Dark-themed control and docking library for .NET WinForms.  
A simple, extensible control library that emulates the look and feel of popular tabbed document interfaces such as Visual Studio, Photoshop, WebStorm, and XCode.

### How to use
The best way to learn how to use DarkUI is to check out the Example project included with the source code. 
It'll show you how to use the majority of the forms, controls, and docking components.  
You can also [read the wiki](https://github.com/MuzzleInteractive/DarkUI.CE/blob/master/Docs/Wiki/README.md). This contains all the information you need to get started as well as more detailed information and guides.
